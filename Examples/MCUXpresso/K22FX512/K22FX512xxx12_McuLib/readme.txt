readme.txt
----------
This is a template for the NXP K22FX512 using FreeRTOS, McuLib and FreeRTOS.
It implements a USB CDC interface.

Note that the McuLib is linked as a linked folder.
The McuLib is shared between different projects.
The location of the McuLib is controlled by the ${MCULIB} Build variable (Project > Properties > C/C++ Build > Build Variables.
