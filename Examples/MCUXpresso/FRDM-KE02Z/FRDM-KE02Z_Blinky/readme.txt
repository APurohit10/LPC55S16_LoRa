readme.txt
----------

Project for the FRDM-KE02Z board with the MCUXpresso IDE and SDK.
It uses a shared McuLib as abstraction from the SDK and hardware (see CDT variable MCULIB).

LEDSs:
- red: PTH1
- green: PTH2
- blue: PTE7
