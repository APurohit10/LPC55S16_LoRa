Project for a tinyK22 board (NXP K22FN512VLH12) with with FreeRTOS to run a 'Hanover Flip Dot' Display.

See https://github.com/hawkz/Hanover_Flipdot for the protocol used.

See 
https://mcuoneclipse.com/2021/06/01/flip-dot-clock-using-used-bus-parts/
for details of the build.
